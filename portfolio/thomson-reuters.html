Objectives & Workplan

Objective


  • Create an audio ID that would be ready for the launch of desktop terminals

  • Find ways to use the mnemonic throughout the Eikon launch, whether on spots, the terminal, or physical location

  • The music and sound design has to be modern, curious and have gravitas

Work Plan


  • Create a palette of instruments that can become as powerful as the four-note audio ID

  • Compose in mixed meters and unusual changes to keep the listener intrigued

  • Using a combination of talent that are from different musical disciplines that create a unique sound, which also becomes as powerful as the audio ID

  • Find ways to use the unique sound and the memorable audio ID in new and creative ways that add value
